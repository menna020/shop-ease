# ShopEase – Online Store (First Project)

**Responsive online store with dynamic shopping cart – built as a first web project to practice HTML, CSS & JavaScript.**

---

## Overview

ShopEase is a simple yet modern and responsive online store layout created as part of my web development learning journey.  
This project showcases a clean design, interactive elements, and a mobile-friendly layout.

---

## Features

- **Responsive Navbar** with a hamburger menu for mobile devices  
- **Professional Shopping Cart** with dynamic item counter  
- **Product Grid** with hover effects and responsive design  
- **Clean and Modern UI** suitable for both desktop and mobile  

---

## Technologies Used

- HTML  
- CSS  
- JavaScript  

---

## Purpose

This project was built to practice building **real-world web layouts** and understanding **responsive design principles**.  
It demonstrates basic **frontend development skills** suitable for beginner portfolios.
